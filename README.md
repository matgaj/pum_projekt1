# Podstawy Uczenia Maszynowego - Projekt 1
Klasyfikacja instrumentów muzycznych z bazy danych [IRMAS](https://www.upf.edu/web/mtg/irmas) algorytmami klasyfikującymi z biblioteki scikit-learn.

Do projektu wykorzystano zbiór IRMAS-TrainingData, który podzielono na podzbiory treningowy i testowy, ponieważ dane z oryginalnych zbiorów treningowego i testowego pozyskane zostały na inne sposoby. Projekt obejmuje klasyfikację dwoma algorytmami, przy dwóch sposobach reprezentacji danych (MFCC oraz openSMILE) oraz optymalizację hipermarametrów przy użyciu biblioteki Optuna.
